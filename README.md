# Pandas Notes

A small learning repo containing structured Pandas practice notebooks covering the basics of Series, DataFrames, and working with CSV files.

## Contents

- `Pandas_Class1.ipynb` — Pandas basics (Series vs DataFrame) + how to read CSV files
- `Pandas_Class2.ipynb` — DataFrames deep dive (inspecting data, selecting rows/cols, index & columns overview, setting columns as index) + mini practice

## Requirements

- Python 3.8+
- Pandas

Install dependencies:

```bash
pip install pandas
```
